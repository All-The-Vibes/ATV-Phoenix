Implement the function format_duration(secs) which converts a whole number of seconds to a compact duration string using hours, minutes, and seconds where 60 seconds is one minute and 3600 seconds is one hour. Rules: omit any unit whose value is zero, for example 3605 seconds gives the string 1h 5s and not the string with a zero minutes; join the present units with a single space; if secs is zero return exactly the string 0s. For example 61 gives 1m 1s. 3600 gives 1h. 120 gives 2m. 3725 gives 1h 2m 5s.
