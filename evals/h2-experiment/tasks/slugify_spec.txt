Implement the function slugify(s) which takes a string and returns a URL slug. It must, in order: first lowercase the string; second remove every character that is not a through z, 0 through 9, whitespace, or a hyphen; third replace every run of whitespace and or hyphens with a single hyphen; fourth strip leading and trailing hyphens. For example the input Hello World gives hello-world. The input a, then three hyphens, then b gives a-b. The phrase Multiple   spaces with extra spaces gives multiple-spaces.
